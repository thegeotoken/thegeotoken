GEOTOKEN

A RESTful API that enables interacting parties to exchange location information in a secure way

Features

  • Tokanized authentication
  • Encrypted data exchange
  • Discrete retention policy (the information is only retrievable witihin 15 min)
  • Uniquely paired keyset bounding exchanging parties

Applications

  • use in conjunction to payment services that are diversifying charges based on location - to add a security layer of your data exchange
  • send media signed with secured tag about occurence location
  • exchange location of an easily compromizable object
  • privately expose location to a selected group or a single party
